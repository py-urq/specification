# specification

## Text output

Print text 

```
p This text will be printed.
p This text will be the same line.
```

Print text with new line

```
pln This text will be printed.
pln This text will be on separate line.
```

## Labels

Label is used to describe locations, actions, menus and procedures.

```
:label
```




## Variables

Set variable

```
apples = 4
```

Variable substitution

```
pln I have #apples$ apples.
```

## Special characters

```
##{code}$ 
```

## Sound

Play music file once

```
play "file_name"
```
